 🐾 JPetStore End-to-End Test Automation Framework

 📌 Overview

This project implements an End-to-End Test Automation Framework for the JPetStore web application using Selenium WebDriver, Java, TestNG, and the Page Object Model (POM) design pattern.

The framework automates critical business workflows and validates the application's functionality across major modules.

---

 🌐 Application Under Test

**Application Name:** JPetStore  
**URL:** https://jpetstore.aspectran.com/

JPetStore is a sample e-commerce web application that simulates an online pet store system.  
It allows users to browse products, add items to cart, and complete checkout.
